### 3个发展方向

* 1.dapp的前后端开发，可能需要补一下区块链交互的部分，其余和web2一样

* 2.协议层开发，偏底层，C++/Go/Java较多，一般涉及到链本身的开发，什么VM编译器最好懂一点，如果本来就是底层开发那么需要补的是密码学基础和共识原理之类的知识

* 3.合约层开发，现在主要是solidity比较多，需要从头学合约的逻辑和开发，虽然比较简单但是技术栈比较独立，同时也要会ts和一些框架，写交互脚本之类的
